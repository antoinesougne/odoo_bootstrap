UX Report

First of all, doing such an exercise is quite difficult without any context. There are a multitude of potential users having radically different needs and problems. Design decisions will vary a lot from one user persona to another.

The best I can do here is to judge if yes or not the design follows common best practices and usability rules. Some of my observations are :

  • The component lacks feedback toward the user. For example, there is no tooltip explaining why the tab is disable. Also, there is no progress visible on the loader.
  • There is a lot to say about the item tabs:
    • It takes a way too big place on the screen
    • There is no indicator telling the user where he is
    • It looks more like a simple list than a clickable element. There is a basic color applied to all the items when it should be an action color for more affordance
  • The color used for the footer makes it stands out too much compared to the main content
